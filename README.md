# hamburg.stefanimhoff.de

This the source of my [travel blog] build with [Eleventy], [PostCSS], and [Gulp].

## Installation

```sh
pnpm install
```

## Running the Website

```sh
pnpm start
```

For more tasks have a look into the `package.json`.
